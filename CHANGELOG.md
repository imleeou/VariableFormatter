## 1.1.0 (2023/10/27)

### Features

- Support for format conversion of words containing spaces.
- **Support for selecting multiple words at the same time and changing the format simultaneously.**

- Add more options
- Support shortcut keys

## 0.0.1 (2023/10/12)

- Initial release
